Gesture-Based Virtual Mouse
A real-time gesture-based human-computer interaction system that allows users to control their computer using hand gestures instead of a mouse.

Core Working
-Webcam captures live video
-Hand is detected using a vision model
-Finger positions are tracked
-Gestures are interpreted
-Corresponding system actions are executed

Technologies Used
-Python
-OpenCV (image processing)
-MediaPipe (hand tracking)
-PyAutoGUI (system control)
-NumPy

Mouse Controls
-Index finger → Move cursor
-Index + Thumb touch → Click
-Index + Middle together → Scroll
